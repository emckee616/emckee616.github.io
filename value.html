How UserBenchmark Affects People's Perception of Value

The way that UserBenchmark ranks parts can very easily result in many people believing that it is always worth paying more for higher spec parts when this is usually not the case. Many expensive parts can be better, but not worth the price. While this does depend on the use of the parts, if someone is using them for playing modern triple-A games, even the best GPU can have trouble with preformance.
